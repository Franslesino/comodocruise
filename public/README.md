# SEASVOYAGE Images

Place your images in the following structure:

```
public/
├── vidfootage.mp4          # Hero section background video
├── images/
│   ├── promo-1.jpg         # Promo section images
│   ├── promo-2.jpg
│   ├── promo-3.jpg
│   ├── promo-4.jpg
│   └── ships/
│       ├── ship-1.jpg      # Ship images
│       ├── ship-2.jpg
│       ├── ship-3.jpg
│       ├── ship-4.jpg
│       ├── ship-5.jpg
│       └── ship-6.jpg
```

All images should be optimized for web (WebP/AVIF recommended)
Recommended sizes:
- Hero video: 1920x1080 or higher
- Promo images: 800x600
- Ship images: 1200x750
